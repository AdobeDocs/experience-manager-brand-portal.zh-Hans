以下图例说明了矩阵中使用的符号：

| 符号 | 含义 |
|--------|-----------------------------------------------------------------------------------------------------------------------------------------------------|
| ✓ | 此文件格式支持此功能。 |
| ✕ | 此文件格式不支持此功能。 |
| NA | 此功能不适用于此文件格式。 |
| * | 在将资产发布到Brand Portal后，此功能需要在AEM创作实例上支持此文件格式，但不需要在Brand Portal上支持此文件格式的附加组件。 |
